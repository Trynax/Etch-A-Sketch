# Etch-A-Sketch
Etch-A-Sketch
Etch-a-Sketch
This project is an implementation of a browser-based version of a classic toy called Etch-a-Sketch using HTML, CSS, and JavaScript. Users can draw on a grid by moving the mouse pointer over it. The project also includes options to reset the grid and change the color of the lines.

Usage
The project can be used by simply opening the index.html file in a web browser. The user can then draw on the grid by moving the mouse pointer over it. To reset the grid, click the "Reset" button. To change the color of the lines, click the color picker and select a new color. To change the size of the grid, click the "Select size" button and enter a new size between 1 and 100 units.
